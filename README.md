## What It Does

Real Estate 3D lets you explore property listings with an interactive 3D room viewer. Browse a property grid, click a listing to open the 3D walkthrough, and orbit around a fully-rendered room with furniture. AI-generated video walkthroughs (via Higgsfield) are planned for the next phase.

---

## Features

- **Three.js room viewer** — rendered floor, walls, ceiling, windows, and furniture (bed, desk, chair, shelving)
- **OrbitControls with auto-rotate** — drag to orbit, scroll to zoom, auto-spins on load
- **Property search + grid** — filterable listing cards with price, beds, baths, and sqft
- **Mock listings** — sample properties wired to the 3D viewer out of the box
- **Higgsfield integration (coming soon)** — AI-generated video walkthroughs for each listing
